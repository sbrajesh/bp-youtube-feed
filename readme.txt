Thanks for downloading Bp-Youtube feed, Hope you will find it Useful.

Installation:-
1.Upload "bp-youtube-feed" to wp-content/plugins
2.Activate Plugin Bp-Youtube from Dashboard->Plugins->Installed->Bp-Youtube
3.Move "yt" folder to your current theme
4.Customize the css as you want, in the yourtheme/yt/style.css

That's it.
